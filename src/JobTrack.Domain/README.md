# JobTrack.Domain

Pure, immutable domain model: interval algebra, hierarchy/achievement, schedule and rate
resolution, and the cost engine. No I/O, no persistence, no ASP.NET Core dependency.

Internal package — part of the JobTrack reusable library, not a standalone published product.
See the [JobTrack repository](https://github.com/lookbusy1344/VSStuff) for the full solution.
