# JobTrack.Persistence.PostgreSql

PostgreSQL implementation of `JobTrack.Application`'s persistence ports using EF Core and Npgsql. `DbContext`, entity configurations, and provider
extensions are internal.

Internal package — part of the JobTrack reusable library, not a standalone published product. See
the [JobTrack repository](https://github.com/lookbusy1344/VSStuff) for the full solution.
